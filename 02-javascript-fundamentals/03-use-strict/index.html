Concise Lesson: “The Modern Mode – use strict”

Main Concepts

Purpose & History

Introduced in ECMAScript 5 (ES5), "use strict" enables a modern mode that enforces stricter JavaScript behavior and catches silent bugs.

References: JavaScript.info, Stack Overflow

How to Enable

Place "use strict"; (or 'use strict';) at the very top of your script or inside a function to apply strict mode to that scope.

References: JavaScript.info, MDN Web Docs

  • It must be the first statement (ignoring comments)—otherwise, it's ignored.
  • Once enabled, strict mode cannot be disabled.

Reference: JavaScript.info

Where It's Automatically Applied

Modern JS constructs like ES6 modules and classes run in strict mode by default—no need to write the directive.
