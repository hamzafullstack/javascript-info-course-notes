Lesson 5: Data Types in JavaScript

JavaScript is Dynamically Typed

Variables can store any type and change type at runtime.

Primitive Types (7)

  • Number – integers & floats. Special values: Infinity, -Infinity, NaN.
  • BigInt – integers beyond Number limit. Written with n at end.
  • String – text inside quotes. Backticks (`) allow embedding ${}.
  • Boolean – true/false values.
  • Null – intentional “empty” or “unknown”.
  • Undefined – variable declared but not assigned.
  • Symbol – unique identifiers.

Non-Primitive

  • Object – collections of data or more complex entities.

typeof Operator

Returns type of a value:

  • typeof 0 → "number"
  • typeof "hi" → "string"
  • typeof Symbol("id") → "symbol"
  • typeof null → "object" (bug in JS)
  • typeof undefined → "undefined"
  • typeof alert → "function"

Concise Summary

JS has 7 primitives (number, bigint, string, boolean, null, undefined, symbol) and 1 main non-primitive (object). typeof helps identify types, but null is misleadingly reported as "object".
