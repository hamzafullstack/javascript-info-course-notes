Q: How do you convert a value to a string?
A: Use String(value) or automatic conversion when outputting.
Q: How do you convert a value to a number?
A: Use Number(value) or automatic conversion in math operations.
Q: What values are falsy in Boolean conversion?
A: 0, "", null, undefined, NaN → false
Q: What values are truthy in Boolean conversion?
A: Everything else (non-empty strings, numbers ≠ 0, objects, etc.) → true
Q: What does Number('123abc') return?
A: NaN because the string is not a valid number.
