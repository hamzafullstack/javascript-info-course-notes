JavaScript Type Conversion Notes

1. String Conversion

Converts a value to a string.

String(value)
Example: 
let val = true;
val = String(val); // "true"

2. Number Conversion

Converts values to numbers, automatically or explicitly.

Automatic in math operations: 
'6' - '2'; // 4
'6' / '2'; // 3
Explicit conversion: 
let str = '123';
let num = Number(str); // 123
Rules:
  • undefined → NaN
  • null → 0
  • true / false → 1 / 0
  • Strings with whitespace → trimmed and converted; empty → 0; invalid → NaN

3. Boolean Conversion

Values converted to true/false.

Rules:
  • Falsy: 0, "", null, undefined, NaN → false
  • Everything else → true
Examples: 
Boolean(''); // false
Boolean('0'); // true
Boolean(1); // true
Boolean(0); // false
Boolean(' '); // true (non-empty string)

4. Summary

  • Three common conversions: String, Number, Boolean.
  • String: output or template literals.
  • Number: math operations or explicit Number(value).
  • Boolean: logical operations or explicit Boolean(value).
  • Always remember conversion rules to avoid NaN or unexpected results.
