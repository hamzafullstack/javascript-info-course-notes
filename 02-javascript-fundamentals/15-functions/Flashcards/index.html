JavaScript Function Basics

Functions are blocks of code designed to perform a particular task. Learn parameters, returns, and defaults step by step.

Flashcards

What is a Function?
A block of code designed to perform a specific task.
Parameters
Inputs passed to a function to work with.
Return
The value a function sends back when executed.
Default Parameters
Set initial values for function parameters.

Descriptions

Functions allow code reusability. Parameters pass data in, return sends results back. Default parameters help when no value is provided.
