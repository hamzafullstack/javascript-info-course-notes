Loops while · do..while · for

Concise theory, essential examples, and flashcards for fast recall.

See notes below.

Core ideas (short)

  • while : check condition before each iteration. Use when iterations depend on runtime condition.
  • do..while : run body once, then check. Use when body must run at least once.
  • for : for(begin; condition; step). Use when iteration count or index control is known.
  • break : exit loop immediately.
  • continue : skip current iteration, continue loop.
  • labels : only use rarely to break outer nested loops (use with care).

Must-remember theory (write these down)

  1. Infinite loops: while(true) or for(;;). Use break to stop.
  2. Single statement bodies may omit braces — but prefer braces for readability.
  3. Loop variable scope: for (let i...) keeps i inside loop.
  4. continue reduces nesting and improves readability vs deep ifs.
  5. Prefer descriptive variable names (index → i ok for simple loops; else use meaningful names).

Essential examples (copy/paste)

while : basic

// count 0..2
let i = 0;
while (i < 3) {
  console.log(i);
  i++;
}

do..while : runs once at minimum

let c = 0;
do {
  console.log(c);
  c++;
} while (c < 3);

for : classic counter

for (let d = 0; d < 3; d++) {
  console.log(d); // 0,1,2
}

skip with continue

for (let n = 0; n < 10; n++) {
  if (n % 2 === 0) continue;
  console.log(n); // 1,3,5...
}

break from nested (label)

outer:
for (let x = 0; x < 3; x++) {
  for (let y = 0; y < 3; y++) {
    if (needToStop()) break outer;
  }
}
