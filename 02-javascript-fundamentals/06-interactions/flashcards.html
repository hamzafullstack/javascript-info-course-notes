Q: What does alert() do?

A: Shows a message to the user and waits for them to press "OK". It is modal.

Q: How does prompt() work?

A: Asks the user to input text. Syntax: prompt(question, default). Returns input or null if canceled.

Q: Whatâ€™s the purpose of confirm()?

A: Shows a message with "OK" and "Cancel". Returns true if OK, false if Cancel.

Q: Are these functions synchronous or asynchronous?

A: Synchronous (modal). Code execution pauses until the user responds.

Q: Can prompt() have a default value?

A: Yes, the second parameter sets the default text. Recommended for IE compatibility.
